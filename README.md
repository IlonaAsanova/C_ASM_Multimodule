# Сброка многомодульных программ. Вычисление корней и определенных интегралов.
Вычисление площади плоской фигуры, ограниченной кривыми, с помощью численных методов: для нахождения приближенных значений вершин фигуры использовался комбинированный метод, для решения основной задачи использовалась квадратурная формула прямоугольников. Программа собирается из трех файлов main.c, externs.c и asmfunc.asm.
